# Security

This project is for **private LANs**. Do not expose the HTTP server to the internet without authentication and TLS.

Report vulnerabilities via [GitHub Security Advisories](https://github.com/ummugulsunn/webos-magic-caster/security/advisories/new) (preferred) instead of a public issue. Include version/commit, reproduction steps, and impact.

Keep Node.js updated; run `npm audit`. Treat `tv-key.json` as sensitive (gitignored).
